# crud-usuarios

## Sobre a Arquitetura.
Minha escolha foi baseada em Clean DDD

# Tecnologias

## Api
* Asp Core 3.1
* Cors
* EntityFrameworkCore
* Swagger 
```
https://localhost:5001/Swagger/
```
* FluentValidation
* Repository Pattern
* DI Pattern

## Front-end
* Angular 12.2.0
* Bootstrap 4.5.3


Executar o script na raiz do projeto: 

```
dotnet ef --startup-project ../Api/ database update
```
